# Dimer example: N2 

To perform the generation of Fermi-orbitals descriptors (FODs) 
from a Lewis structure use following command
```bash
bash run.sh run.py 
```

You can analyze the produced geometry for instance using 
ase gui 
```bash 
ase gui *_fods.xyz 
```

You can clean the directory using 
```bash
bash clean.sh  
```
